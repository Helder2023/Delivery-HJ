Importar dados

Importação para Adminstradores

Quando clicar no botão , o sistema fará upload de um ficheiro CSV (Excel) que irá inserir todas as informações actuais no banco de dados.

Este ficheiro contém todas as actividades do utilizador: dispositivo usado, IP, álbuns, faixas, pagamentos, entre outras.

Atenção: A importação é feita para todos os utilizadores. Certifique-se de que os dados estão totalmente actualizados antes de importar! Se tiver dúvidas, leia este artigo na Central de Ajuda.
