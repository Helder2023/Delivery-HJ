Política de Privacidade

A Política de Privacidade tem como objetivo informar os funcionários sobre a coleta, uso, armazenamento e tratamento de seus dados pessoais pelo sistema de Dashboard Wasom Upfy.

Dados coletados

São coletado pelo sistema os dados pessoais dos funcionários quando estes, fazem bakcups no sistema, gerenciam os utilizadores, actividades, interagem com os utilizadores de qualquer forma.

Os dados coletados podem incluir, nome completo, endereço de e-mail, senha, data de nascimento, sexo, localização e entre outros dados.
